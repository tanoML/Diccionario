Diccionario de palabras


El primer archivo que debe ser abierto es el llamado index.html
luego al ingresar lo primero que podriamos ver es una presentacion
de los equipos algo de historia etc. etc.

Posteriorimente si observamos en la barra de navegacion encontraremos tres enlaces
Inicio, Palabras, lista y Agregar palabras que se explican a continuacion:

Inicio: Enlace a la pagina principal cuando se navega a otro lado.
Palabras: Se propone que sea donde se muestre las palabras de forma individual con sus respectivas imagenes.
Lista: Se propone que se muestre solo las palabras pero en forma de tabla, y se pueda ir recorriendo respectivamente.
Agregar palabra: Basicamente es donde se podran agregar las palabras a la base de datos.

Esta es la version de interfaces, aun falta etapas de implementacion y pruebas para finalmente desplegarlo en etapa de produccion.
